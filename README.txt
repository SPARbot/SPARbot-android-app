# SPARbot-android-app

Android application with Video Calling, VR and ROS packages to operate the bot.


1. android-video-calling

Implemented modules:
* Video calling over the internet.
* Person to person call establishment.

Video implementation:

https://drive.google.com/file/d/0B3hP9XdjvDqgbzFuTFBwaW5EYzQ/view?usp=sharing


2. android-virtual-reality

Android application to provide VR experience to the user. This application communicates with the Raspberry Pi 3 to receive the live camera feed from RasPi 3 camera and display a stereoscopic image to the user on the mobile.
Google Cardboard SDK is used to achieve VR experience on the Android application.
