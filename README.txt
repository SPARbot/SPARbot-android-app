# Virtual Reality


Android application to provide VR experience to the user. This application communicates with the Raspberry Pi 3 to receive the live camera feed from RasPi 3 camera and display a stereoscopic image to the user on the mobile.
Google Cardboard SDK is used to achieve VR experience on the Android application.
